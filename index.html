    Modules for Legend of the Green Dragon

    Project maintained by wmzaru Hosted on GitHub Pages — Theme by mattgraham

    Modules for version 1.x.x - Modules, Skins and third party support files for version 1.x.x. These are NOT compatible with version 0.9.7

    Administrative Modules - Additional features and improvements to the game administration tools.

    Clan Modules - Additional mods for Clans

    Dwellings - Modules for the Dwelling system

    Forest Events - Special events for the forest

    Forest Modules - Additional modules for forest activities, not including special events.

    German Language (Deutsch) - Modules written in German.

    Lodge Points - Assorted modules for use in the lodge (donator points)

    LoGD Game Add-Ons - Non modularized add-ons for the LOGD game.

    Miscellaneous - Modules and files which couldn't be pigeonholed elsewhere can be found here.

    Other Languages - Modules written in languages other than English and German

    Patches - Game modifications which require alteration of core code.

    Races - Additional races

    Shades Modules - Additional modules for the Shades (Graveyard), including special events.

    Specialties - Additional Specialties

    Templates - Skins aka Themes aka Templates provide different looks for your game.

    Travel Events - Special Events specifically for Travel

    Unsupported Core Modifications - Modified core files, ostensibly to extend functionality. These files should only be used by experienced admins, and are UNSUPPORTED by DragonPrime. Use at your own risk.

    Village Add-Ons - Village features, such as Shops and other places to visit or see.

    Village Events - Special Events for the Village

    Vital Stats Display - Modules which add to the player's vital stats display.
